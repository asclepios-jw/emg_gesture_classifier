# EMG Gesture Classification

This repository contains a complete deep learning pipeline for classifying gestures using raw EMG (Electromyography) sensor data. 

We process a massive dataset of 4.2 million time-series rows into structured sliding windows and use a 1D Convolutional Neural Network (1D-CNN) built with PyTorch to classify the signals into 7 different gesture classes. The model achieved a final accuracy of **96.08%**.

## ­¤ōü Project Structure

- `preprocess.py`: Loads the raw `EMG-data.csv`, filters out noise (`class == 0`), extracts overlapping sliding windows (size: 200, step: 50), applies Z-score normalization, and saves the datasets as `.npy` files.
- `train.py`: Loads the preprocessed `.npy` files, builds the 1D-CNN architecture, handles the training loop (with Adam, Weight Decay, and ReduceLROnPlateau), and evaluates the best model.
- `report.txt`: The final classification report generated by `train.py`.
- `confusion_matrix.png`: A visual confusion matrix of the final test evaluation.

*(Note: Large files such as `EMG-data.csv` and `.npy` arrays are ignored via `.gitignore` to keep the repository lightweight.)*

## ­¤ÜĆ How to Run

### 1. Install Dependencies
Ensure you have Python 3 installed. Install the required libraries:
```bash
pip install pandas numpy scikit-learn torch matplotlib seaborn
```

### 2. Prepare the Data
Download or place the `EMG-data.csv` dataset in the root directory of this project.

### 3. Preprocessing
Run the preprocessing script to generate the windowed datasets (`X_train.npy`, `X_test.npy`, `y_train.npy`, `y_test.npy`).
```bash
python3 preprocess.py
```

### 4. Training and Evaluation
Run the training script. This will train the CNN model for up to 40 epochs and automatically generate `report.txt` and `confusion_matrix.png` at the end.
```bash
python3 train.py
```

## ­¤¦Ā Model Architecture

The classifier uses a 1D Convolutional Neural Network (CNN) specifically tuned for 8-channel time-series data:

1. **Conv Block 1**: Conv1d (128 filters, kernel=5) -> BatchNorm -> ReLU -> MaxPool -> Dropout(0.3)
2. **Conv Block 2**: Conv1d (256 filters, kernel=5) -> BatchNorm -> ReLU -> MaxPool -> Dropout(0.3)
3. **Conv Block 3**: Conv1d (512 filters, kernel=3) -> BatchNorm -> ReLU -> MaxPool -> Dropout(0.3)
4. **Classifier**: Flatten -> Linear (1024) -> ReLU -> Dropout(0.5) -> Linear (7 output classes)

## ­¤ōŖ Results

The optimized architecture paired with heavy regularization (Dropout & Weight Decay `1e-4`) prevents overfitting and allows the model to generalize excellently, reaching an overall test accuracy of **96.08%**.
